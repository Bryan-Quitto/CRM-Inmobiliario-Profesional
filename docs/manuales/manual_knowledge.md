# Manual de Base de Conocimiento Institucional

## 1. Gestión de Preguntas Frecuentes (FAQs) por Propiedad

En este sistema, las Preguntas Frecuentes (FAQs) son exclusivas para cada propiedad y se encuentran dentro de la pantalla de detalles de cada inmueble. Su función principal es enseñarle a la Inteligencia Artificial los detalles específicos de la propiedad para que pueda responder a los clientes. 

### Ciclo de vida y Moderación

- **Creación y Borradores:** Cualquier usuario puede escribir nuevas preguntas frecuentes para una propiedad y guardarlas como borrador hasta que estén listas, o borrarlas si ya no sirven.
- **Flujo de Revisión:** Cuando terminas de redactar una pregunta, debes enviarla a revisión. El agente responsable de esa propiedad evaluará el contenido antes de que el Asistente Inteligente pueda usarla para responder.
- **Aprobación o Rechazo:** El agente responsable revisa las preguntas sugeridas y decide si las aprueba (para que la IA las use inmediatamente) o si pide correcciones. Solo las preguntas aprobadas alimentan al asistente.
- **Edición y Mantenimiento:** Si una pregunta ya está aprobada o fue devuelta para correcciones, siempre se puede editar para mejorar o actualizar la información que recibe la IA.
- **Activación y Desactivación:** Si una pregunta frecuente ya no aplica por un tiempo, puedes desactivarla para que la IA no la use temporalmente, y volver a activarla cuando sea oportuno.

## 2. Documentos y Conocimiento de la Empresa

Además de las preguntas por propiedad, el sistema permite subir documentos generales de tu empresa.

- **Subida de Documentos:** Puedes subir archivos con información importante de la empresa. El sistema los analizará y guardará para que el Asistente Inteligente pueda usarlos para responder dudas de forma precisa.
- **Procesamiento Automático:** Al subir un documento, el sistema hace el trabajo pesado de analizarlo silenciosamente por detrás para que puedas seguir trabajando sin interrupciones.

## 3. ¿Dónde encontrar estas herramientas?

- **Preguntas por Propiedad:** Puedes crear y gestionar las preguntas frecuentes específicas de cada inmueble directamente mientras ves los detalles de la propiedad. ¡Todo está a la mano!
- **Base de Conocimiento de la Empresa:** Desde el panel de Configuración, en la sección de Inteligencia Artificial, puedes subir archivos de texto y decidir si esa información será solo para uso interno de tu equipo, para el público general, o para ciertas agencias.
